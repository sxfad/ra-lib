## 组件容器

## API

| 属性 | 说明 | 类型 | 默认值 |
| --- | --- | --- | --- |
| loading | 显示loading | `boolean` | `false` |
| loadingTip | 显示loading是提示信息 | `string 或 ReactNode` | `加载中...` |
| fitHeight | 适应窗口高度，内容过长，Content会产生滚动条 默认 false | `boolean` | `false` |
| otherHeight | 计算高度时，额外的高度，默认0，content会撑满全屏 | `number` | 0 |
